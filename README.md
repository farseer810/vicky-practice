####  000, 给整数a, b，计算两数的和与积

输入：共一行，两个数字以空格隔开
输出：第一行输出a+b的和，第二行输出a*b
```python
if __name__ == "__main__":
    """
    line = input() # 读一行字符串
    a, b = line.split(' ') # 以空格分离一行字符串
    a, b = int(a), int(b) # 转换成整数类型
    """
    a, b = input().split(' ') # 读取一行字符串并以空格分开
    a, b = int(a), int(b)
    print(a + b)
    print(a * b)
```
